# 3D Endless Runner Game (Unity)
A fast-paced, performance-optimized 3D action game built with Unity 6, 
featuring procedural world generation and interactive gameplay mechanics.
# Project Overview
This project is inspired by popular runner titles like Subway Surfers. 
The player navigates an infinite track, avoids obstacles through lane-switching, 
jumping, and sliding, while collecting coins to unlock character skins. 
# Technical ImplementationProcedural 
# Track Generation: 
Implemented a modular tile system that recycles tracks dynamically to create an infinite environment without runtime lag.
# Object Pooling:
Used for coins and track tiles to minimize memory overhead and improve performance on mobile/PC devices.
# Lane-Based Movement:
Responsive character controller for jumping, sliding, and rapid lane switching using Unity’s physics and animation systems. 
# Skin Customization:
A modular system (SaveSkinChange & ChangeSkin) that allows players to use collected coins to purchase and apply different character skins. 
# Key Gameplay FeaturesPower-Ups:
Includes a Flight Ability that allows the player to temporarily fly over obstacles.
# Progression System:
Real-time scoring and persistent coin collection for rewards. 
# Interactive UI:
Complete game loop with a Start Menu, HUD (Heads-Up Display), Customization Screen, and Game Over states.
# Audio Feedback: 
Integrated sound clips for actions like jumping, collecting coins, and collisions to enhance immersion.
